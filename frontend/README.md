# Quiz Extractor — Frontend

Next.js 14 application. See the [root README](../README.md) for full setup instructions.

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # production build
```

Requires `NEXT_PUBLIC_API_URL` in `.env.local` pointing to the backend.
